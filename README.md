# Dee’s Store — E-commerce Frontend

A modern e-commerce storefront built with React as a real-world side project for a small business.

## This project focuses on:

 - Clean UI/UX design

 - Reusable component structure

 - Centralized theming with CSS variables

 - Responsive layout patterns

## Tech Stack

 - React (Create React App)

 - CSS (custom theme system + reusable base styles)

 - Ionicons

## Current Features

 - Animated header navigation

 - Product showcase cards with overlay content

 - Centralized design system for consistent styling

## Goals

 - Practice real-world frontend architecture

 - Build scalable UI components

 - Create a production-style storefront layout

## Getting Started

```bash
npm install
npm start
```

### Notes

> This project is a learning-focused side build created to gain hands-on experience while helping a small business get started online.
---
> Built as a practical frontend development project to demonstrate component design, theming, and UI structure.
